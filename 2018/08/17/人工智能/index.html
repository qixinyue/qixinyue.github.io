Numpy的数组

1 numpy中的数组是ndarray类型的对象
将实际数据和元数据分开存放，独立操作，以此提高性能。
元数据就是对实际数据的描述，比如姓名 性别 年龄等来描述一个人，实际数据就是描述所指向的实际的人。
修改元数据的指向关系，实际数据不改变，这样就可以做到修改了顺序但是数据并没有真正的修改，从而提高性能。

2 numpy数组元素类型必须相同–同质性

3 numpy数组的元素可以通过基于0 的下标单独访问，
size个元素的数组，合理的下标范围 [0~size-1]或者[-size,-1](闭区间）

4 numpy数组通过dtype和shape属性表示元素的类型和纬度，其中纬度的类型为元组，按照从高到低的顺序排列每一维的大小
比如数组为

1 2 3
4 5 6

则 shape：(2,3) 2行 3列
dtype为int32 有符号的32位整型数据
字符串的dtype为 <UX,X是数组中最长的字符串的长度， <代表小端字节序 U代码unicode编码
如下数组

1 2 3

4 5 6

7 8 9

shape：(3,1,3) 3页，每页1行，每行3个元素

5 数组的创建的方法：
numpy.arange(start=0,end,step=1) start起始值，缺省默认为0 end终止值必须有 step步长默认为1
numpy.array(literable) literable是可迭代对象，是任意可以被解释为数组的序列

 ___________________________

  源数组.ravel()->一维目标数组

实际数据 <- 非共享 -> 实际数据副本

import numpy as np
import matplotlib.pyplot as mp

x = np.linspace(-np.pi, np.pi, 1000)

cos_y = np.cos(x) / 2
sin_y = np.sin(x)

xo = np.pi / 2
yo_cos = np.cos(xo) / 2
yo_sin = np.sin(xo)

mp.xlim(x.min() * 1.1, x.max() * 1.1)  # 留一点空白区域
mp.ylim(sin_y.min() * 1.1, sin_y.max() * 1.1)
mp.plot(x, cos_y, linestyle='-', color='orangered', label=r'$cos(x)$')

          [r'$-\pi$', r'$-\pi/2$', r'$0$', r'$-\pi/2$', r'$-\pi$'])

ax = mp.gca()

ax.spines['right'].set_color('none')

        color='green', label=r'$sin(x)/2$')

加权平均值

样本 s = [s1,s2,…sn]
权重 w = [w1,w2,…wn]
加权平均值 a = (s1w1 + s2w2 +… sn*wn)/sum(w)
numpy.average(s,weights=w) -> a
算术平均值就是各个样本权重相等时的加权平均值

成交量加权平均值
代码如下：

import numpy as np
closing_price, volumes = np.loadtxt(
    './data/aapl.csv',
    delimiter=',',
    usecols=(6, 7),
    unpack=True,  # 对列进行展开
    dtype=np.dtype('f8,f8'),  # 运行后类型为datetime64[D]
)

vwap = 0
for cp, vol in zip(closing_price, volumes):
    vwap += cp * vol
vwap /= volumes.sum()
print(vwap)

vwap = np.average(closing_price, weights=volumes)
print(vwap)

中位数

将多个样本按照大小顺序排列，居于中间位置的元素即为中位数
奇数个元素就是中间的，如果是偶数个元素则求最中间的2个算术平均数作为中位数
L:序列长度
M：(A[(L-1)//2]+A[L//2])/2 得到中位数
numpy.median(A) -> M
代码：

import numpy as np

标准差

样本 a = [5.0,5.1,5.1,4.9,4.9]
算术平均值为 m = 5
离差 D= [a1-m,a2-m…]
离差方： Q=[(a1-m)^2,(a2-m)^2,…]
方差: v = sum((a[i]-m)^2)/a.size
标准差计算方式
std = sqrt(v) = sqrt( ((5.0-5.0)^2 + (5.1-5.0)^ 2 + (5.1-5.0)^ 2 +(4.9-5.0)^ 2 +(4.9-5.0)^ 2)/5 )
^代表幂运算

numpy.std(a,ddof=非自由因子) -> std 非自由因子 默认为0，如果为1，则表示n-1个样本是自由的
代码：

import numpy as np

closing_price = np.loadtxt(
    './data/aapl.csv',
    delimiter=',',
    usecols=(6),
    unpack=True,  # 对列进行展开
    dtype=np.dtype('f8'),  # 运行后类型为datetime64[D]
)

mean = closing_price.mean()  # 平均值
devs = closing_price - mean  # 离差
devs2 = devs ** 2  # 离差方
pvar = devs2.mean()  # 总体方差
svar = devs2.sum() / (devs2.size - 1)  # 样本方差

sstd = np.sqrt(svar)  # 样本标准差
pstd = np.sqrt(pvar)  # 总体标准差
print(sstd, pstd, sep='\n')

import datetime as dt

    wday = date.weekday()
    return wday

weeks, opening_prices, highest_price, lowest_price, closing_price = np.loadtxt(
    './data/aapl.csv',
    delimiter=',',
    usecols=(1, 3, 4, 5, 6),
    unpack=True,  # 对列进行展开
    converters={1: dmy2wday}
)

weeks = weeks[:16]
opening_prices = opening_prices[:16]
highest_price = highest_price[:16]
lowest_price = lowest_price[:16]
closing_price = closing_price[:16]

first_monday = np.where(weeks == 0)[0][0]  # 第一个星期一
last_friday = np.where(weeks == 4)[0][-1]  # 最后一个星期五
indoces = np.arange(first_monday, last_friday + 1)  # 下标

indoces = np.split(indoces, 3)
print(indoces)


def week_summary(indoces):
    # 返回每周的开盘价 最高价 最低价 收盘价
    open_price = opening_prices[indoces[0]]
    high_price = np.max(np.take(highest_price, indoces))
    low_price = np.min(np.take(lowest_price, indoces))
    close_price = closing_price[indoces[-1]]
    return open_price, high_price, low_price, close_price


summaries = np.apply_along_axis(week_summary, 1, indoces)
print(summaries)
np.savetxt('./data/summary.csv', summaries, delimiter=',', fmt='%g')
